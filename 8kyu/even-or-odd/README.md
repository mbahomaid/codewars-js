# Even or Odd

## Description

Create a function that takes an integer as an argument and returns `"Even"` for even numbers or `"Odd"` for odd numbers.

## Example

```js
evenOrOdd(2); // "Even"
evenOrOdd(7); // "Odd"
evenOrOdd(0); // "Even"
```

## Requirements

- Input: An integer (`number`)
- Output: A string, either `"Even"` or `"Odd"`
